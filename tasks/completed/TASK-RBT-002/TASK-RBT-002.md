# Task: Fix python-library template

## Description

The python-library template at `~/.agentecflow/templates/python-library/` was generated by `/template-create` from the `youtube-transcript-mcp` source project, but the AI analysis misidentified the language as JavaScript. The manifest.json, settings.json, and CLAUDE.md all contain incorrect JavaScript metadata. This task fixes all metadata to accurately reflect a Python library template.

## Changes Required

### manifest.json (CRITICAL — complete rewrite of key fields)

1. `display_name`: "Javascript Layered Architecture" -> "Python Library"
2. `description`: -> "Python pip-installable library with hatchling, src layout, pytest, ruff, mypy strict"
3. `language`: "JavaScript" -> "Python"
4. `language_version`: null -> ">=3.10"
5. `frameworks`: [] -> add hatchling, pytest, ruff, mypy
6. `architecture`: "Layered Architecture" -> "Src Layout Library"
7. `category`: "general" -> "library"
8. `layers`: ["Infrastructure", "Shared", "Application", "Handlers"] -> ["src", "tests", "docs"]
9. `tags`: Remove all JavaScript tags, add python, library, hatchling, pytest, ruff, mypy, src-layout, pip
10. Remove `source_project` (hardcoded absolute path)
11. Set `placeholders.Author.default_value` to null
12. Recalculate/update `confidence_score` (currently 68.33 — reflects wrong detection)

### settings.json (CRITICAL — complete rewrite)

1. `naming_conventions`:
   - `class.case_style`: "PascalCase" (correct, keep)
   - `class.suffix`: ".ts" -> ".py"
   - `function.case_style`: "camelCase" -> "snake_case"
   - `function.examples`: Remove lunr.js examples
   - Add `module` convention: snake_case, ".py"
2. `code_style`:
   - `indent_size`: 2 -> 4
   - `line_length`: 100 (keep, matches ruff config from spec)
   - Add `type_hints`: "required"
   - Add `docstrings`: "google_style"
3. `file_organization`:
   - Keep `by_layer: true`
   - `test_location`: keep "separate"
4. `layer_mappings`: Rewrite all entries:
   - Remove Infrastructure, Shared, Handlers (JavaScript layers)
   - Add: src (source code), tests (test files)
   - Fix `file_patterns` from `["*.js", "*.jsx"]` to `["*.py"]`
   - Remove hardcoded `youtube_insights_mcp/services` path
